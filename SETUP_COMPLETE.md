### 📦 Installation Documented

**Backend Setup**
```bash
cd backend
npm install
npm start
# Runs on http://localhost:5000
```

**Frontend Setup**
```bash
cd frontend
npm install
npm run dev
# Runs on http://localhost:5173
```

**AI Server Setup**
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python run_ai_server.py
# Runs on http://localhost:8000
```
